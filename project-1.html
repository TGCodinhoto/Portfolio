Cardápio Online

Projeto de uma hamburgueria com o cardápio online, onde é possível selecionar os itens que deseja e colocá-los no carrinho!

Link do projeto

Projeto

Um projeto criado em conjunto com SujeitoProgramador, onde foi desenvolvido um cardápio online para uma hamburgueria que possibilita selecionar itens e coloca-los no carrinho para efetuar o pagamento com o somatório final.

Este projeto foi feito usando tecnologias como: HTML, CSS, JS e Tailwind. É um projeto responsivo, ou seja, se adapta a qualquer tamanho de tela!

Tecnologias usadas

HTML
CSS
JavaScript
Tailwind
